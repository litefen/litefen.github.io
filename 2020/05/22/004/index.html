修改MIUI12状态栏背景图片手机端电脑端都可以操作- 小爱同学自定义音色,pcm文件替换

2020年05月22日 603 字 大概 2 分钟

此教程分为两个阶段
第一阶段：开启小爱同学的自定义音色功能
第二阶段：替换音色包

开启自定义音色功能（这里给出两种方法）

小白用户建议用第一种方法非常简单

第一种方法

此方法不更改小爱同学应用，在原来的小爱同学音色设置里面是看不到音色设置的录制，每次新录制需要从快捷方式打开，想要在小爱同学a音色设置里面看到可以用方法二（需要改代码）
下面开始操作方法一
下载创建快捷方式APP链接

打开它
点开右上角三点
勾选下面三项

在搜索框搜索小爱同学 如图
然后点开活动列表

看图示操作
打开小爱同学的活动列表之后
搜索框继续搜索def
找到图中款选的那一项点开详情

点击打开

然后就如下图
也可以给这个应用创建快捷方式的权限，创建音色录制的快捷方式，每次从桌面快捷方式打开就可以音色录制

以上是第一种方法不需要改代码的方法

第二种方法

需要MT管理器 酷安搜索即可，还是给个链接吧MT管理器
下面开始操作
用MT管理器找到下列目录
/data/data/com.miui.voiceassist/app_resource_updated/
打开vendor_tts_config.json这个json文件

修改代码（不懂语法的就别乱动按照图示操作就可以了，少个空格多个点都会出问题）
{
“vendor_tts_model”: [
“机型代号”
],
“asr_detect_score”: 0.4,
“total_feature_on”: true
}


修改完保存点右上角保存，重新打开小爱同学，就可以在小爱同学音色设置里面看到可以用音色录制功能了
你们可能会在录制的时候遇到提示环境过于嘈杂，请选择安静的录制环境的问题
可以插拔耳机解决（我是这么解决的）
